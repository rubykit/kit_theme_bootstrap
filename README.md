# KitThemeBootstrap

[KitThemeBootstrap] is a gem that contains [Kit] default bootstrap theme.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kit_theme_bootstrap'
```

And then run:

```bash
$ bundle install
```
